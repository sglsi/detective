# 华生表情集资源清单

## 资源概述

华生（Dr. John Watson）游戏内表情集，共 18 种情绪神态，用于对话系统和角色互动。

## 角色设定

- **年龄**：29岁（1852年出生）
- **背景**：军医，刚从阿富汗回来
- **外观**：整洁八字胡，眼神沉稳
- **服饰**：深灰长款西装外套、棕色格纹马甲、米白衬衫+领结、深棕长裤、黑色皮鞋
- **道具**：烟斗、怀表与表链

## 技术规格

- **尺寸**：512×512 像素
- **格式**：JPEG
- **风格**：维多利亚时代 sepia 色调
- **构图**：头部及肩部特写

## 游戏内使用建议

### 1. 对话系统
- 按场景需求裁剪为单张头像（约 100×100 ~ 150×150）
- 导入 Godot 作为 `Texture2D` 资源
- 与 `08_血字的研究_对话台词库.md` 中各场景的 mood/emotion 字段对应

### 2. 提示系统
- `watson_思考.jpg` / `watson_凝思.jpg`：用于 L2 华生提示
- `watson_赞同.jpg` / `watson_倾佩.jpg`：用于玩家正确推理时的反馈
- `watson_疑惑.jpg`：用于玩家卡住时的引导

### 3. 角色图鉴
- 可作为图鉴解锁后的展示大图
- 建议配合 UI 框架中的深棕褐（#3A2416）和羊皮纸色（#F5ECD5）背景使用

## 界面功能拆解

### 1. 案件主界面（Blackwood Mansion 风格）

该图可作为**游戏核心 HUD 与推理墙界面**的视觉参考，包含：

- **顶部标题栏**：案件名称 + 福尔摩斯剪影 + 设置按钮
- **左侧导航栏**：案件 / 人物 / 思维殿堂 / 证物 / 报纸 —— 对应 08_系统框架设计中的 `NotebookManager` 与 `InventoryManager`
- **中央地图区**：伦敦真实地图 + 红色线索连线 + 5 个关键节点标记 —— 对应推理墙的可视化推理链（06_推理墙运行机制.md）
- **右侧线索卡片**：线索标题 + 描述 + 观察记录 + 操作按钮（查看/关联/推理/记录）— 对应 15 字段线索数据结构（03_可执行侦探游戏关卡设计稿.md）
- **底部操作栏**：调查 / 推理 / 询问 / 追踪 / 文档 —— 对应场景交互模式

> **注意**：该界面图展示的「Blackwood Mansion」为示例案件标题，实际项目中应替换为「血字的研究」相关素材。

### 2. 选项设置界面

- 左栏：LOOK / TALK / EXAMINE / THINK / JOURNAL / SAVE / LOAD —— 对应场景侧栏快捷操作
- 右区：游戏设置、音频设置、语言设置 —— 对应 `SettingsManager` 与 `AudioManager`
- 该图可复用为**设置弹窗或暂停菜单**的视觉模板

### 3. 对话交互界面

- 左侧动作按钮：观察 / 对话 / 检查 / 思考 / 日志 / 保存 / 读取
- 中央场景视图：伦敦街景 + 大本钟，符合维多利亚时代背景
- 右侧 NPC 头像 + 对话气泡 —— 与 `福尔摩斯_表情集` 中的表情资源配合使用
- 底部福尔摩斯立绘 + 对话文本 —— 对应 08_血字的研究_对话台词库.md

### 4. 伦敦市地图界面

- 1891 年伦敦市大地图，包含主要地标：
  - 贝克街 221B
  - 大英博物馆
  - 威斯敏斯特宫
  - 圣保罗大教堂
  - 伦敦塔
  - 白金汉宫
  - 圣潘克拉斯车站
  - 摄政公园
- 左栏：侦探笔记 / 线索分析 / 人物关系 / 案情推理 / 更换包装 —— 对应导航与案件进度
- 右下角：委托信 —— 对应案件触发与任务目标
- 与 `01_血字的研究_地点考证规划.md` 中考证的地点坐标可直接对照

### 5. 主标题界面

- 标题：Sherlock Holmes / The Case of the Crimson Letter
- 背景：伦敦地图 + 大本钟 + 福尔摩斯剪影 + 放大镜/怀表/报纸道具
- 按钮：开始游戏 / 选项 / 退出游戏
- 对应 `06_主界面设计方案.md` 中的主菜单设计

> 实际项目中应替换为「血字的研究」或中文标题，并调整中文排版。

### 6. 难度选择界面

- 三张维多利亚风格卡片并排：入门侦探（EASY）/ 正式探案（NORMAL）/ 大师挑战（HARD）
- 入门侦探卡带有金色高亮引导光效
- 黄铜边框 + 深色羊皮纸底纹 + 金色文字
- 维多利亚装饰元素：煤气灯、放大镜图标、华丽边框
- 对应 `difficulty_select.tscn` 场景和 `DifficultyManager` 系统

### 7. 六步探索闭环界面

- 六边形/环形流程图：观察发现 → 工具操作 → 数据记录 → 知识检索 → 假设形成 → 验证修正
- 每步配备维多利亚风格图标（放大镜/镊子/羽毛笔/书本/齿轮/天平）
- 蒸汽朋克黄铜机械连接器串联各步骤
- 当前激活步骤以金色边框高亮
- 对应 `GameScene` 的 13 状态 GamePhase 状态机

### 8. 线索记录与分析界面

- 维多利亚风格证据卡片：线索标题（金色衬线字体）+ 手写体描述 + 观察笔记 + 分析结果面板
- 操作按钮：查看 / 关联 / 推理 / 记录（黄铜按钮风格）
- 左侧导航栏：案件 / 人物 / 证物 / 报纸
- 放大镜和指纹装饰元素
- 对应线索五状态机（UNDISCOVERED→DISCOVERED→RECORDED→ANALYZED→LINKED）

### 9. 推理墙/思维殿堂界面

- 中央：维多利亚时期伦敦地图，红线连接线索节点形成推理网
- 浮动线索卡片通过红线连接，类似调查板风格
- 左侧：证据库存栏，右侧：推理链面板
- 华丽维多利亚边框、深色软木板纹理、黄铜图钉、手写注释
- 对应 `ReasoningWallManager` 三层五区架构

### 10. 案件结算/星级评定界面

- 顶部：三维评价体系——观察力 / 推理力 / 洞察力，各带星级评分
- 中部：七枚徽章以维多利亚勋章风格展示（锐眼 / 推理大师 / 深度探求 / 完美评分 / 速通达人 / 无提示精通 / 初案告破）
- 底部：案件统计摘要 + 继续按钮
- 深色天鹅绒背景、金色星星、黄铜框架、暖色庆祝灯光
- 对应 `StarRatingSystem` 和七徽章系统

## 使用建议

1. **作为风格参考**：这些图完整定义了维多利亚侦探游戏的视觉风格（深棕褐、金色、羊皮纸、旧报纸、金属边框），建议作为所有 UI 设计的统一参考。
2. **作为界面布局模板**：可直接在 Godot 中按此布局搭建 `Control` 节点层级。
3. **作为切图资源**：需要时可将按钮、边框、标题、图标等元素切出为独立 PNG，作为 9-slice 或普通纹理使用。
4. **作为宣传图**：高清版（1536×1024）适合作为宣传或 pitch 展示素材。

## 后续处理建议

1. **分辨率统一**：当前图片有 1536×1024 与 565×376 两种比例，实际项目需统一为 1920×1080 或 16:9 目标分辨率。
2. **切分 UI 元素**：标题、按钮、边框、图标、线索卡片框等可单独切出。
3. **中文化适配**：图片中的英文标题和按钮需要替换为中文，保持维多利亚风格字体。
4. **动态化改造**：静态图片需拆分为 Godot 的 `TextureRect` + `Button` + `Label` 等节点，实现交互逻辑。

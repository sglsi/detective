### 场景一 对话台词 <a id="scene-1"></a>

#### 阶段1：初次见面 —— 悬念开场

[简单模式|普通模式|困难模式]  {

福尔摩斯（上下打量华生，停顿2秒，特写，自信、从容）："……阿富汗军医。"

华生（一愣，特写，吃惊表情）："什么？"

福尔摩斯："我说，你是一名刚从阿富汗回来的军医。我说对了吗？"

华生（惊讶）："您……您怎么知道的？我们刚认识不到十秒钟。"

福尔摩斯（微笑，转向玩家视角）：
  "这位新朋友显然不相信。不如——你来告诉他我是怎么看出来的？"

【系统提示】新手教程：第一次观察
→ 目标：找出4条线索，证明"华生是阿富汗军医"
→ 操作提示：移动视角，观察华生身上的细节
}

#### 阶段1 教程环节：六步探索闭环（点击华生）

**设计说明**：本环节为玩家的第一次完整六步探索闭环体验。
Step 1-3 为必做（MVP核心），Step 4-5 为可选（功能展示），Step 6 为必做收束。
首次自然引入：侦探笔记、知识库、推理墙、三星评价、结论里程碑。

福尔摩斯（可选触发，玩家点击条目时）：

（收集2个以上碎片后，出现可选入口：「去推理墙建个假设？」）

[首次进入引导]  {
【系统提示】欢迎来到推理墙——把线索串起来，就是推理。
极简引导：拖拽两条线索到中间的假设板上 → 形成初步假设

[MVP · 全难度通用]  {
假设状态仅三态：待建立 / 活跃中 / 已确认
玩家拖拽"刚从热带回来"+"军人气质" → 生成假设："可能是热带地区的军人？"
假设标记为「活跃中」
福尔摩斯："在有全部证据之前就下结论，是最大的错误。但我们可以先有个猜测。"
}

[M2+扩展]  {
假设状态五态：待建立 / 活跃中 / 领先中 / 已确认 / 已排除
支持多假设并行竞争
支持自由连线和自由猜想
}
}

 Step 6：验证修正（必做 · MVP四级验证）
━━━━━━━━━━━━━━━━━━━━━━

（收集全部4个碎片后，引导玩家回到推理墙验证假设）

[简单模式]  {
系统提示："线索集齐了！去推理墙验证你的结论吧"
推理墙中，4条线索自动排列，玩家只需确认连线方向

玩家完成连线 → 系统展示验证结果：
  【四级验证】证据充分 · 已确认（VERIFIED）
  → 刚从热带回来 + 左臂负伤 + 面容憔悴 + 军人气质
  → 结论：阿富汗军医

【里程碑点亮动画】"初识推理" 解锁
【获得】观察之星 ⭐

福尔摩斯（满意，特写）：
  "你看，四条线索指向同一个结论。这就是回溯推理的力量——
  从结果倒推原因，把碎片拼成一幅完整的画。"
}

[普通模式]  {
系统提示："线索收集完毕，去推理墙验证你的假设"
玩家需自行拖拽连线、组织推理链

验证算法（MVP纯计数法）：4条线索全部用上 → VERIFIED
少于4条 → SUPPORTED（证据基本支持）
2条以下 → INSUFFICIENT（证据不足）

验证结果为VERIFIED时：
  【里程碑点亮】"初识推理" 解锁
  【获得】观察之星 ⭐
  福尔摩斯："不错的推理。你已经摸到门道了。"

验证结果低于VERIFIED时：
  福尔摩斯："方向是对的，但还有更关键的细节你漏掉了。再回去看看？"
  → 允许玩家返回Step 1/2重新探索（不扣分，可推翻重做）
}

[困难模式]  {
无验证提示，玩家自行判断何时"证据充分"
玩家主动提交验证 → 系统给出四级结果
错误可推翻重做，不惩罚

验证通过后：
  【里程碑点亮】"初识推理" 解锁
  【获得】观察之星 ⭐
  福尔摩斯（微微点头）："还不赖。"
}

#### 阶段1 教程后对话

[简单模式|普通模式（概率触发提示）]  {

华生（目瞪口呆，特写，敬佩表情）：
  "这……太不可思议了。仅仅从这些细节，就能看出这么多？"

福尔摩斯（看向玩家，略带欣赏）：
  "不错的初次表演。要知道，大多数人看到的只是表象，
  而侦探看到的是——历史。"

福尔摩斯："介绍人说过你现在的职业是一名医生。那么，
  军人 + 医生 + 热带 + 负伤 = 阿富汗军医。这不是什么魔法，
  只是系统化的观察和推理。"

华生："系统化……听着像一门科学。"

福尔摩斯（微笑）："它就是一门科学。只是大多数人不懂得运用。"
}

[普通模式（概率未触发）|困难模式]  {

华生（有点疑惑，但明显被说服）：

华生："听起来……你经常做这样的事？"

福尔摩斯："这是我的工作。准确地说——是我的爱好。"
}

【系统提示】推理墙已开启，随时可查看当前推理链

赫德森太太端着茶盘进来："福尔摩斯先生，茶来了。哦，华生医生，欢迎您。"

#### 阶段2 信使观察环节（第二次六步闭环练习）

**设计说明**：难度递增的第二次练习。相比阶段1：
- 线索从4个增至6个（含2个干扰项）
- 首次引入时间压力（调查进度计时）
- 首次展示四级验证的不同结果对话
- 验证等级不同 → 福尔摩斯回应不同，增加 replay value
【第二次观察练习开始】

━━━━━━━━━━━━━━━━━━━━━━
 Step 1-3：观察→工具→记录（流程同阶段1，不再赘述）
━━━━━━━━━━━━━━━━━━━━━━

6个可观察点：
  ① 手背蓝色大锚文身 ✓（正确：海员特征）
  ② 络腮胡 ✓（正确：军人式胡须）
  ③ 昂首挺胸的站姿 ✓（正确：军人气概）
  ④ 自信发号施令的神态 ✓（正确：军士气质）
  ⑤ 袖口磨损 ✗（干扰：只是衣服旧了）
  ⑥ 走路略跛 ✗（干扰：只是今天脚扭了）

[简单模式]  {
4个正确线索高亮闪烁，2个干扰项灰暗标注"无关"
全程引导，工具自动推荐，笔记自动填充
无时间压力

验证结果：必为VERIFIED
福尔摩斯（满意）："很好，你已经掌握了基本方法。"
}

[普通模式]  {
6个观察点微光提示，不区分正确/干扰
工具自选，选错有温和提示
笔记需玩家自行滑杆标记
70%概率触发中途提示："袖口那个磨损……可能只是穿久了。"（对应hint_base_probability=0.7）
30%概率出现误导性强调："等等，他走路好像有点跛？"（对应mislead_probability=0.3）
无时间压力

验证结果根据玩家选择动态判定：
  4个正确 + 0个干扰 → VERIFIED（已确认）
  3个正确 ± 干扰 → SUPPORTED（基本支持）
  2个正确 ± 干扰 → INSUFFICIENT（证据不足）
  1个及以下 → CONTRADICTORY（与结论矛盾）

VERIFIED时福尔摩斯："不错。你已经学会区分信号和噪音了。"
SUPPORTED时："方向对了。但有些无关的东西，你还得学会过滤。"
INSUFFICIENT时："还不够。再仔细看看——关键的特征就在眼前。"
}

[困难模式]  {
无任何提示，6个观察点需玩家自行发现
工具自选，选错无反馈
笔记完全空白，自由记录
70%概率混入深度干扰项：信使口袋露出半张药房收据（看似线索，实际只是感冒药）
无时间压力，全凭玩家判断

验证结果同上四档，判定更严格
VERIFIED时福尔摩斯（微微点头）："还行。"
其余等级不主动评价，玩家可求助触发提示（HelpSystem · M2+）
}

 Step 6：验证修正（四级结果展示）
━━━━━━━━━━━━━━━━━━━━━━

[全难度通用]  {
玩家提交验证 → 系统展示四级验证结果 + 对应星级评价
→ 观察之星累计（第二颗星的潜在获得条件）

【系统提示】四级验证体系说明（首次展示）
  · 已确认 VERIFIED — 证据充分，结论可靠
  · 基本支持 SUPPORTED — 方向正确，证据尚不足
  · 证据不足 INSUFFICIENT — 线索太少，无法支撑结论
  · 矛盾 CONTRADICTORY — 现有证据与结论冲突

验证通过（≥SUPPORTED）：
  → 解锁案件委托信阅读
  → 侦探笔记自动归档本次观察结果

验证失败（INSUFFICIENT / CONTRADICTORY）：
  → 允许返回重探，不扣分，可推翻重做
  → 福尔摩斯："别急着下结论。回去再看看。"
}

#### 阶段2 信使环节收尾对话

[简单模式]  {

信使（听完玩家推理，惊奇）："太对了！您也像福尔摩斯先生一样厉害！"

福尔摩斯（微笑）："这位是我的新搭档。观察和推理，不是什么魔法，只是方法。"

信使（谦逊中略微骄傲）："哈哈，军营里的一些习惯，到现在还改不掉。"

福尔摩斯："不用改。好习惯是侦探最好的老师。让我们先看看这封信吧。"
}

[普通模式（概率触发提示）|简单模式]  {

福尔摩斯（对玩家）："注意到了吗？关键的不是你看到了多少，
  而是你能从里面挑出多少真正有用的。"

    福尔摩斯戴上手套，拿起药丸，仔细观察，药丸为珍珠似的灰色，小而圆，迎着亮光看简直是透明的。
    葛莱森警长一屁股坐在椅子里："卡彭蒂耶目前还在警局，看来这个小伙子不可能牵连到第二个谋杀案里了，福尔摩斯，看来你的推断是正确的。"
    福尔摩斯（略带骄傲）："当然了，从现场留下来的足迹就足以推断出嫌疑人的体型和卡彭蒂耶相差很大。当然，这也必须到他家看到他本人或照片后才能确定。"
    }

#### 阶段4：药丸实验解密

    [简单模式|普通模式|困难模式]  {
    福尔摩斯将取出一粒，把它切成两半，把半粒放入试管，加入试剂。
    试管中没有反应。
    "这不可能是偶然的，绝不可能仅仅是由于巧合。在德雷伯一案中我猜到会有某种药丸，现在这种药丸在斯特兰森死后真的被发现了。但是它们竟然不起作用。究竟是怎么回事？肯定地说，我的一系列推论绝不可能出现谬误！绝不可能！但是这个试剂居然没有反应。"
    福尔摩斯一边大声说，一边站了起来。"在屋里情绪烦躁地走来走去，哦，我明白了！我明白了！"他突然高兴地尖叫了一声，跑到药盒前，取出另外一粒，把它切成两半，把半粒溶在水里，产生剧烈反应，溶液变为黑色。
    福尔摩斯长长地吁了一口气，擦了擦额头上的汗珠："我还不够自信。刚才我就应当意识到，如果一个情节似乎和一系列的推论相矛盾，那么，这个情节必定有某种其他解释。那个小匣里的两粒药丸，一粒是烈性的毒药，另外一粒则完全无毒。其实在我没看到这个小盒子之前，就应该想到的。"
    福尔摩斯："药丸为某种生物碱。这是从一种南美洲土人制造毒箭的毒药中提炼出来的，毒性非常猛烈，只要沾到一点儿，立刻就能置人死地。"
    雷斯垂德："所以德雷伯是被毒死的？"
    福尔摩斯："是的，我在推理凶手的心思。这个凶手一定在德雷伯面前提出了'上帝惩罚'——两颗药丸，一颗有毒，一颗没毒，让德雷伯选择。德雷伯选中了毒药，死了。而斯特兰森不愿意，所以被直接刺死。"
    }

§18 对话台词库（条件分支）—— 场景八：起居室（最终对决/真相揭示）

### 场景八 对话台词 <a id="scene-8"></a>

#### 阶段1：布局抓捕凶手

    [简单模式|普通模式|困难模式]  {
    [门被敲响]
    门外有人敲门——原来是街头流浪儿的代表，那个不起眼的小维金斯来了。
    维金斯举手敬礼说："先生，请吧，马车已经喊到了，就在下边。"
    福尔摩斯和蔼地说道，"好孩子！"
    福尔摩斯一边说，一边微笑了起来，"很好，很好。最好让马车夫帮我搬箱子。去叫他上来，维金斯。"
    房间里有一只小小的旅行皮箱，福尔摩斯把它拉了出来，开始系上面的皮带。当他正在忙着的时候，马车夫走了进来。
    "车夫，帮我扣好这个皮带扣。"福尔摩斯曲膝在那里摆弄着皮箱，头也不回地说。
    这个家伙紧绷着脸，不大愿意地走向前去，伸出两只手正要帮忙。说时迟，那时快，只听到钢手铐咔嗒一响，福尔摩斯突然跳起身来。
    福尔摩斯的双眼闪烁着光芒，"先生们，让我给你们介绍杰弗森·霍普先生，他就是杀死德雷伯和斯特兰森的凶手。"
    这只是一刹那的事，福尔摩斯脸上的胜利表情，他那响亮的语声以及马车夫看着闪亮的手铐像魔术似的一下子铐上自己手腕时那种茫然、凶蛮的面容；马车夫愤怒地大吼一声，挣脱了福尔摩斯的控制，向窗子冲去，把木框和玻璃撞得粉碎。但是，就在他正要钻出去的时候，葛莱森、雷斯垂德和福尔摩斯就像一群猎狗似的一拥而上，把他揪了回来。一场激烈的战斗开始了，这个人凶猛异常，我们四个人一再被他击退。他似乎有着一股疯子似的蛮劲儿。他的脸和手在跳窗时割破了，一直在流血，但是他的抵抗并未因此减弱。直到雷斯垂德用手卡住他的脖子，使他透不过气来，他才明白挣扎已无济于事。就是这样，我们还不能放心，于是又把他的手和脚都捆了起来。捆好了之后，我们才站起身子，不住地喘着气。
    福尔摩斯微笑着说，"他的马车在这里，就用他的马车把他送到苏格兰场去吧。好了，先生们，这件小小的神秘莫测的案子，总算可以告一段落了。现在，我欢迎各位提出任何问题，绝不会再拒绝答复。"
    }

#### 阶段2：凶手完整自白

    [简单模式|普通模式|困难模式]  {
    福尔摩斯、葛莱森、雷斯垂德和华生一起押送杰弗森·霍普回到警局交给值班警官。
    值班警官是个面色白皙、神情冷淡的人，他机械而呆板地履行着他的职务。他说："犯人将在本周内提交法庭审讯。杰弗森·霍普先生，你在审讯之前，还有什么话要说吗？我必须事先告诉你，你所说的每一句话都要记录下来，并且可能作为定罪的依据。"
    杰弗森·霍普平静地说："我也许永远不会受到审讯了。我得了主动脉瘤症！上个星期，我找一位医生看过，他对我说，过不了多少天，血瘤就要破裂了。这个病已经好多年了，一年比一年恶化。这个病是我在盐湖城的大山之中，由于饱经风霜，过度操劳，而且又吃不饱的缘故引起的。现在我已经完成了自己的工作，什么时候死，我都不在乎了。但是，我愿意在死之前，把这件事交代明白，死后好有个记载。我不愿在死后让别人把我看成一个寻常的杀人犯。"
    值班警官说："如果是这样的话，为了维护法律，显然，我们的职责是首先取得他的口供。先生，你现在可以自由交代了。不过，我再一次告诉你，你所交代的内容都要记录下来。"
    杰弗森·霍普："我已经是坟墓边上的人了，所以不会对你们说谎的。我所说的每一句话，都是千真万确的。至于你们如何处置，对我来说，就无关紧要了。"
    杰弗森·霍普："我为什么要恨这两个人，他们恶贯满盈，他们犯了罪，害死过我最亲密的两个人——一个父亲和一个女儿，因此他们付出了自己的性命，也是罪有应得的。是的，他们有罪！我打定主意，要把法官、陪审员和刽子手的任务全部一个人承担起来。如果你们是男子汉大丈夫，如果你们站在我的位置上，你们也一定会像我这样干的。"
    杰弗森·霍普："我刚才说的那个姑娘，二十年前本来是要嫁给我的，可是却被迫嫁给了德雷伯，以致含恨而死。我从她遗体的手指上把那只结婚戒指取了下来，当时就发誓，一定要让德雷伯看着这只戒指毙命，还要在他临死的时候，让他认识到，是由于自己犯下的罪恶，才受到了惩罚。我踏遍了两大洲，追踪着德雷伯和他的帮凶；一直到追上他们为止，这只戒指都一直带在我身边。我在这个世界上的工作已经完成了，而且是出色地完成了。他们两个人已经死了，是被上帝惩罚而死的，除此之外，我再也没有什么别的希望和要求了。"
    杰弗森·霍普："一天傍晚，当我赶着马车在他们所住的那个叫陶尔魁里的地方徘徊的时候，忽然看见德雷伯和斯特兰森上马车而去。我赶紧催马加鞭，远远跟在他们后面。他们到了尤斯顿车站，下了马车，走进了月台。我听到他们打听去利物浦的火车，站上的人回答说，有一班车刚刚开出，几个钟头之内不会再有第二班车了。斯特兰森听了之后，似乎很懊恼，可是德雷伯却非常高兴。德雷伯说，他有一点儿私事要去办一下，如果斯特兰森愿意等他的话，马上就会回来。他的伙伴却阻拦他，并且提醒说，他们曾经决定过时时刻刻都要在一起，不能单独行动。德雷伯回答说，这是一件微妙的事，他必须独自前往。我听不清斯特兰森又说了什么，后来只听见德雷伯破口大骂，并且说，斯特兰森不过是自己雇用的仆役罢了，不要装腔作势地对他发号施令。于是，这位秘书先生讨了一场没趣，只好不再多说。他只是和德雷伯商量，万一耽误了最后的一班火车，可以到郝黎代旅馆去找他。德雷伯回答说，他在十一点钟之前就可以回到月台上来。然后，他就一直走出了车站。"
    杰弗森·霍普："他在路上走着，并且进了一家酒店，停留了半个钟头，出来的时候，步履蹒跚，显然已经醉了。在我的前面恰好有一辆双轮小马车（汉索姆hansom的双轮小马车，车夫坐在车的最后面） ，他就招呼着坐了上去。我一路紧紧地跟着。但使我感到诧异的是，我们竟然又回到了他原来住的地方。我想象不出，他回到那里究竟要干什么。"
    杰弗森·霍普："我等了一刻钟，或者还要久一点儿，突然房子里面传来一阵打架似的吵闹声。接着，屋门忽然大开，出现了两个人，其中一个是德雷伯，另一个是个年轻的小伙子，这个小伙子一把抓住德雷伯的衣领，当他们走到台阶边的时候，他用力一推，紧跟着又是一脚，把德雷伯踹到了大街当中。他对德雷伯摇晃着手中的木棍，大声喝道：'狗东西！我教训教训你，你竟敢污辱良家妇女！'他是那样怒不可遏，如果不是这个坏蛋拖着两条腿拼命向街中跑去，我想，那小伙子一定会用棍子把他痛打一顿。德雷伯一直跑到转弯的地方，正好看见了我的马车，于是招呼着我，一脚就跳上车来。他说：'把我送到郝黎代旅馆去。'"
    杰弗森·霍普："我一见他坐进了我的马车，简直喜出望外，心跳得非常厉害。我真怕就在这个千钧一发的时刻，自己的血瘤迸裂了。这时，他的酒瘾又发作了，叫我在一家大酒店的外面停了下来。他一边吩咐我等着他，一边走了进去。他在里面一直待到酒店关门，等出来的时候，已经是烂醉如泥了。我知道，自己胜券在握了。"
    杰弗森·霍普："当时午夜已过，快一点钟了。这是一个凄风苦雨的深夜，风刮得很厉害，大雨倾盆而下。外面虽然是一派惨淡的景象，可是我的心里却乐不可支，高兴得几乎要大声欢叫起来。诸位先生，如果你们之中哪位曾经为一件事朝思暮想，盼望了二十多年，一旦伸手可得，那么，你们就会理解到我当时的心情了。当我赶着马车前进时，看见老约翰·费里尔和可爱的露茜在黑暗中望着我微笑。我看得清清楚楚，就像现在在这间屋子里看见你们诸位一样。一路之上，他们总是在我的前面，一边一个地走在马的两旁，一直陪着我来到布瑞克斯顿路的那所空宅。"
    杰弗森·霍普："我想，他以为已经到了刚才提到的那个旅馆，因为他什么话也没有说，就走下车来，跟着我进了空屋前的花园。这时，他还有点儿头重脚轻，站立不稳，我不得不扶着他走，以免跌倒。我们走到门口时，我开了门，引着他进了前屋。我敢向你们保证，一路上，费里尔父女一直在我们前面走着。"
    杰弗森·霍普："我擦燃了一根火柴，把带来的一支蜡烛点亮。转向他，把蜡烛举近了自己的脸。对他说，好啦，伊诺克·德雷伯，现在你看看我是谁！"
    杰弗森·霍普："他醉眼惺忪地盯着我看了半天。然后，我看见他的脸上突然出现了恐怖的神色，整张脸都痉挛起来，这说明他已经认出了我。他吓得面如土色，摇摇晃晃地后退着，大颗的汗珠从额头滚落到眉毛上。他的牙齿也在上下相击，咯咯作响。我看见他这副模样，不禁靠在门上大笑不止。我早就知道，报仇是一件最痛快的事，可是，从来没有想到竟会有这样的快感。"
    杰弗森·霍普："你这个狗东西！我把你从盐湖城一直追到圣彼得堡，可是总让你逃脱。现在你东躲西藏的日子终于到头了，因为，不是你就是我，再也见不到明天的太阳了。我说话的时候，他又向后退了几步。我从他的脸上可以看出，他以为自己发疯了。那时，我确实和疯子一样，太阳穴上的血管像被铁匠挥舞着铁锤敲打似的跳动不止。我深信，当时若不是血从鼻孔中涌了出来，使我轻松一下的话，我的病也许就会发作起来了。"
    杰弗森·霍普："你说，露茜·费里尔现在怎么样了？惩罚确实来得太慢，不过现在总算让你落网了。我看到在我说话的时候，他那两片怯懦的嘴唇颤抖着，还想求饶。但是，他心里很清楚，这是毫无用处的了。"
    杰弗森·霍普："他叫道，杀死她和她父亲的并不是我。"
    杰弗森·霍普："但是，你粉碎了她那颗纯洁的心！我厉声喝道，把毒药盒子送到他的面前，来让上帝裁决吧。挑一粒吃下去。一粒可以致死，一粒可以获生。你挑剩下的一粒我吃。让咱们瞧瞧，世界上到底还有没有公道，或者说，咱们都是在碰运气。"
    杰弗森·霍普："他吓得躲到一边，大喊大叫起来，哀求饶命。但是，我拔出刀，直抵他的咽喉，一直到他乖乖地吞下了一粒，我也吞下了另外一粒。我们面对面，不声不响地站在那里有一两分钟之久，等着瞧究竟谁死谁活。当他的脸上显出痛苦表情的时候，就知道了自己已吞下毒药。他当时的那副嘴脸我怎么能够忘记呢？我看他那副样子，不觉大笑起来，并把露茜的结婚戒指举到他的眼前。这一切只是一会儿工夫，因为那种生物碱的毒性发作得很快。一阵痛苦的痉挛使他的脸都扭曲变形了。他两手前伸，摇晃着，然后惨叫一声，一头倒在了地板上。我用脚把他翻转过来，用手摸了摸他的心口。心不跳了，他死了！"

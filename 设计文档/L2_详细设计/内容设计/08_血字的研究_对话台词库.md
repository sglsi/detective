## §18 对话台词库（条件分支）—— 全部 8 个场景

**教学渐进说明**：
- 场景一（贝克街221B）：全引导 —— 六步闭环完整展示，每步有明确引导
- 场景二（劳瑞斯顿花园室外）：半引导 —— Step 1-3 有提示，Step 4-6 需自主
- 场景三（劳瑞斯顿花园室内）：轻引导 —— 仅 Step 1 有提示，其余自主
- 场景四~八：自主探索 —— 无引导，玩家完全自主运用已学技能

### 场景一 对话台词 <a id="scene-1"></a>

#### 阶段1：初次见面 —— 悬念开场

[EASY 模式|NORMAL 模式|HARD 模式]  {

福尔摩斯（上下打量华生，停顿2秒，特写，自信、从容）："……阿富汗军医。"

华生（一愣，特写，吃惊表情）："什么？"

福尔摩斯："我说，你是一名刚从阿富汗回来的军医。我说对了吗？"

华生（惊讶）："您……您怎么知道的？我们刚认识不到十秒钟。"

福尔摩斯（微笑，转向玩家视角）：
  "这位新朋友显然不相信。不如——你来告诉他我是怎么看出来的？"

【系统提示】新手教程：第一次观察
→ 目标：找出4条线索，证明"华生是阿富汗军医"
→ 操作提示：移动视角，观察华生身上的细节
}

[NORMAL 模式]  {
系统提示："线索收集完毕，去推理墙验证你的假设"
玩家需自行拖拽连线、组织推理链

验证算法（MVP纯计数法）：4条线索全部用上 → VERIFIED
少于4条 → SUPPORTED（证据基本支持）
2条以下 → INSUFFICIENT（证据不足）

【VERIFIED 时】：
  【里程碑点亮】"初识推理" 解锁
  【获得】观察之星 ⭐
  【徽章检查】首次通关 → FIRST_CASE_CLEAR 候选

【SUPPORTED 时】：
  福尔摩斯："方向是对的，但还有更关键的细节你漏掉了。再回去看看？"
  → 允许玩家返回Step 1/2重新探索（不扣分，可推翻重做）

【INSUFFICIENT 时】：
  福尔摩斯（微微皱眉）："仅凭一两条线索就下结论？你还需要更多观察。回到起点，重新看看华生。"
  → 允许返回重探

【CONTRADICTORY 时】：
  福尔摩斯（摇头）："你的结论和你观察到的证据自相矛盾。一个刚负伤的人不可能同时是现役军人。重新想想。"
  → 允许返回重探
}

[HARD 模式]  {

【VERIFIED 时】：
  【里程碑点亮】"初识推理" 解锁
  【获得】观察之星 ⭐
  【徽章检查】HARD模式通关 → NO_HINT_MASTER 候选
  【徽章检查】首次通关 → FIRST_CASE_CLEAR 候选
  福尔摩斯（微微点头）："还不赖。"

【SUPPORTED / INSUFFICIENT / CONTRADICTORY 时】：
  福尔摩斯不主动评价，玩家可求助触发提示（HelpSystem · M2+）
  允许返回重探
}

**【视角一：福尔摩斯】**

**【视角二：杰弗森·霍普】**
（画面切换至街对面巷口的阴影中）
杰弗森·霍普的红脸膛隐没在帽檐下的阴影里，只有那双眼睛在昏暗中闪烁着冷峻的光。
  他看着福尔摩斯跳上马车后部，看着马车缓缓启动，看着那个叼着烟斗的身影渐渐消失在街道尽头。

#### 阶段2：询问房东卡彭蒂耶太太 —— 六步闭环

[EASY 模式|NORMAL 模式|HARD 模式]  {
葛莱森警长："卡彭蒂耶太太，你好，你们应该听到你们以前的房客，克利夫兰城的德雷伯先生被人谋杀的消息了吗？"

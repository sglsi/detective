## 二、主界面设计方案

### 2.1 设计定位

主界面是玩家进入游戏后的**第一个视觉印象**，需要达成三个目标：
1. **氛围传达**：一秒进入维多利亚伦敦的探案世界
2. **功能清晰**：核心操作一目了然（开始/继续/设置等）
3. **视觉统一**：与游戏内界面风格无缝衔接，不做割裂
**核心设计理念**：主界面不是"菜单页"，而是**贝克街221B的书桌**——一张铺满案件资料的旧书桌，侦探刚刚起身，烟斗尚有余温，等待玩家落座续查。
参考图已高度还原此概念，本方案以参考图为基准，调整项目适配细节。

**// gdscript
**# MainMenu.gd
extends Control

@onready var continue_button = $ButtonLayer/ContinueButton
@onready var save_info = $ButtonLayer/ContinueButton/SaveInfo

func _ready():
    # 存档检测
    var has_save = SaveManager.has_saved_game()
    continue_button.visible = has_save
    if has_save:
        save_info.text = SaveManager.get_latest_save_summary()

    # 进场动画
    $EntranceAnimation.play("entrance")

    # 待机动画启动
    $InvestigationProps/Magnifier/LensReflection.play("reflection")
    $ForegroundProps/PocketWatch/WatchHand.play("tick")
    $CharacterLayer/SmokeParticles.emitting = true

func _on_ContinueButton_pressed():
    SaveManager.load_latest_game()
    SceneManager.transition_to("GameMainInterface", "zoom_out")

func _on_NewCaseButton_pressed():
    SceneManager.transition_to("CaseSelection", "page_flip")

func _on_CasebookButton_pressed():
    SceneManager.transition_to("CasebookInterface", "page_flip")

func _on_OptionsButton_pressed():
    $SettingsPanel.popup_centered()

func _on_QuitButton_pressed():
    get_tree().quit()

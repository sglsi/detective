 # SOUL

 一、项目使命
 让玩家以福尔摩斯的第一视角，身临其境地体验 19 世纪维多利亚时代伦敦的推理探案过程，在游戏中感受经典文学的魅力，同时通过真实历史地图和严谨的推理逻辑，获得沉浸式的动态小说体验。

 二、核心愿景
 成为全球最受欢迎的福尔摩斯主题推理游戏，兼具文学价值、历史真实性和游戏趣味性，为不同水平的玩家提供渐进式的推理体验，并通过跨平台同步和案件库复玩机制，持续吸引玩家探索。

 三、核心价值观
 1. 忠实原著，沉浸体验
 以《福尔摩斯探案全集》为蓝本，还原经典案件的推理逻辑
 第一视角扮演福尔摩斯，感受大侦探的侦察思维和技能
 通过对话和场景观察双线驱动故事，打造动态小说般的沉浸感

 2. 历史真实，严谨考证
 基于 1864 年斯坦福图书馆版伦敦真实地图，所有地点均经坐标考证
 还原维多利亚时代的社会背景、人物对话风格和侦破工具
 推理逻辑符合真实逻辑链，多重证据交叉验证

 3. 渐进开放，友好包容
 分阶段发布：网页游戏优先，降低玩家进入门槛，零注册即可体验
 支持游客模式与注册系统，满足不同玩家的需求
 三种难度模式：简单（引导式）、普通（概率提示）、困难（硬核推理），覆盖休闲到核心玩家

 4. 跨平台同步，持续进化
 网页端与移动端共用同一后端，数据完全一致，进度无缝同步
 案件库架构：忠实原著案件入库，积累后可随机组合，增加复玩性
 遵循稳定性与可扩展性原则，避免底层重构，支持持续功能扩展

 四、设计哲学
 地图不是工具，而是收藏品；游戏不是关卡，而是动态小说。

 玩家来体验的是破案的爽感，而非研究 19 世纪伦敦地理
 地图随案件进度逐步解锁，最终成为玩家独有的收藏品
UI 设计以沉浸优先，可隐藏界面元素，最大程度展现游戏内容

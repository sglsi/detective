# 福尔摩斯像素艺术角色资源

## 概述
维多利亚时代福尔摩斯侦探的像素艺术角色资源，适用于 Godot 4.x 游戏引擎。

## 角色特征
- **年龄**：35 岁左右
- **外貌**：英国人特征，消瘦棱角分明的脸型，高挺鹰钩鼻
- **眼神**：神秘、智慧、琥珀色眼睛
- **服装**：棕色猎鹿帽、棕色格纹因弗内斯斗篷、深色马甲、白色硬领衬衫、深色领带

## 文件结构
```
pixel_art/
── sherlock_spritesheet.png    # 完整精灵表（含动画帧）
├── README.md                    # 本说明文件
└── animations/                  # 分割后的动画帧
    ├── idle/                    # 待机动画（5 帧）
    ├── walk/                    # 行走动画（5 帧）
    ├── think/                   # 思考动画（5 帧）
    ├── inspect/                 # 检查动画（5 帧）
    └── point/                   # 指向动画（5 帧）
```

## 技术规格
- **帧尺寸**：64x64 像素
- **格式**：PNG（支持透明背景）
- **调色板**：16 色
- **引擎兼容**：Godot 4.x

## 动画说明
| 动画名称 | 帧数 | 用途 |
|---------|------|------|
| idle | 5 | 待机状态 |
| walk | 5 | 行走移动 |
| think | 5 | 思考/推理 |
| inspect | 5 | 使用放大镜检查 |
| point | 5 | 指向/指示 |

## 表情资源
表情文件位于：`assets/portraits/pixel/`

包含 14 种表情：
- 喜悦/开心/兴奋/自信/坚定
- 疑惑/沉默/思考/凝思/疲惫
- 生气/愤怒/神秘/狡黠

## 使用示例
```gdscript
# 在 Godot 中使用 AnimatedSprite2D
var sprite = $AnimatedSprite2D
sprite.play("idle")
sprite.play("walk")
sprite.play("think")
```

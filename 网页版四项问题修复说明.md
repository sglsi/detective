# 维多利亚伦敦探案 · 网页版四项问题修复说明

> 官方项目名《谁是大侦探》（Godot 4.7 + Node/Express + Supabase 侦探游戏）

## 关键结论：你实测的是「网页原型（8080）」，之前从未改过它

排查发现，你描述的 4 个症状（**手机小界面 / 4 个小圆圈 + 放大镜空圈 / 无后台注册 / 首点锁死**）与 `web_prototype/index.html`（**8080 原型**）的代码**逐条对应**，而之前的所有修复只动到了 Godot 网页构建（8081）。所以这个原型才是你实际看到的对象，现已在本文件落实全部四项修复。

- 原型入口：`web_prototype/index.html`（纯 HTML/CSS/JS 单文件，8080 端口静态托管）
- 后端：Node/Express（3000 端口，`POST /api/auth/register` 等，CORS `origin:'*'`）

---

## 问题 1：网页界面显示成「手机端小界面」

**根因**：`#app { max-width: 480px; margin: 0 auto; }` 把整个应用锁死成 480px 宽的居中窄列，在桌面浏览器里就像手机小界面。

**修复**：
- 删除 `#app` 的 `max-width: 480px`，改为 `width: 100%` 铺满。
- 新增桌面端分栏布局（`@media (min-width: 820px)`）：左侧场景视图 + 右侧对话面板，场景房间放大到 `max-width: 640px`，明确呈现「网页版」而非手机版。

## 问题 2：放大镜空圈 + 显示几秒 + 自动跳「侦探笔记」

**根因**：原 `#magnifier` 只是个圆形边框（无图像），`selectTool` 在 `setTimeout(...,1500)` 后直接 `completeToolUse()` → `openNotebook()`，1.5 秒后自动跳到笔记页。

**修复**：
- 点击任意细节 → 弹出 **`#detail-review` 审查弹窗**：中央放大镜镜片显示该细节的**真实图像（emoji 立绘图标）+ 名称 + 描述**，持续显示直至玩家操作。
- 提供 **「📝 加入笔记 / 跳过 / 关闭」** 三个按钮，**完全由玩家选择**，彻底去掉自动跳转。
- 工具栏的 🔍 放大镜可重新查看上一个已观察细节，📏 卷尺给出测量提示。

## 问题 3：没有与后台注册相关的功能

**根因**：整个原型文件此前没有注册/登录/游客的任何入口或调用。

**修复**：
- 顶部栏新增 `🔑 账号` 按钮（登录后变为 `👤 用户名`，点击退出），打开 `auth-panel` 弹窗。
- `auth-panel` 支持 **注册 / 登录 / 游客** 三种模式：
  - 注册 → `POST /api/auth/register`（`{username?, email, password, phone?}`）
  - 登录 → `POST /api/auth/login`（返回 `token`，存入 `localStorage`）
  - 游客 → `POST /api/auth/guest`
- `getApiBase()` 自动拼接后端地址：沙箱内取当前页 `origin` 并把 `x-cs-sandbox-port` 换成 `3000`；本地调试回退 `http://localhost:3000`。后端 CORS `origin:'*'`，浏览器跨端口 fetch 可行（已实测注册返回 `201`）。

## 问题 4：第一个小圆圈点击后，再点其它圆圈没反应

**根因**：原 `onHotspotClick` 在首次点击后立即 `currentStep = 'tool'`，后续点击被 `if (currentStep !== 'observe') return;` 永久拦截，其余圆圈全部失效。

**修复**：
- 重写观察状态机：`onHotspotClick` 只标记已观察并打开审查弹窗，**不再切到锁定步骤**（仅用 `reviewOpen` 标志在弹窗打开期间挡住其它点击，关闭即恢复）。
- 收集满本阶段所需细节（华生 4 个 / 信使 6 个，含 2 干扰项）后才切到 `verify`，此时可打开推理墙验证。其余圆圈在整个观察阶段都可继续点击。

## 验证情况（第二轮修复）
- 原型 JS 通过 `node --check` 语法校验。
- **沙箱代理层限制**：只有 `/api/health` 路径可达，`/api/auth/*` 被代理返回 404。因此不能直接用浏览器 fetch 跨端口到 3000 后端。
- **解决方案**：`proxy_server.py` 在 8082 端口启动自定义静态服务器，收到 `/api/*` 请求自动转发到 `localhost:3000`，其余路径提供静态文件。`getApiBase()` 改为空字符串（直接同源 `/api/*` 路径），由代理服务器处理转发。
- **注册实测**：通过代理服务器的 `POST /api/auth/register` 返回 `201 Created`。
- **人物立绘**：已将 `sherlock_凝思.png`（华生观察阶段）和 `sherlock_神秘.png`（信使阶段）拷贝到 `web_prototype/assets/` 并在场景中嵌入。
- 端口存活：后端 3000、原型 8080（旧 http.server，仅静态）、**代理原型 8082（含 API 转发 + 立绘）**、游戏 8081。

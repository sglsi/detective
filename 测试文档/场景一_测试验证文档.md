## 4. L3: 信号链路测试

### 4.1 六步闭环信号链路

```
DialogueManager.step_entered(step, name)
  → GameScene._on_step_entered(step, name)
    → current_phase 切换
    → tool_bar.show_toolbar() / reasoning_wall.open()
    → _show_notification()

验证: 每个 Step 1-6 的信号发射→接收链路
```

### 4.2 热点交互信号链路

```
玩家点击热点按钮
  → SceneController._on_hotspot_button_clicked(id)
    → _on_hotspot_clicked(id)
      → ClueEventBus.clue_discovered(id)
        → GameScene._on_clue_discovered(id)
        → ReasoningWallUI._on_clue_discovered(id)
      → SceneEventBus.hotspot_clicked(id)
        → GameScene._on_hotspot_clicked(id)
          → _handle_phase1_hotspot(id)
            → tool_bar.show_toolbar()
```

### 4.3 验证完成信号链路

```
玩家点击"验证推理"按钮
  → ReasoningWallUI._on_verify_pressed()
    → ClueEventBus.verification_complete(result)
      → GameScene._on_verification_complete(result)
        → VERIFIED: _handle_verified() → PHASE1_COMPLETE
        → SUPPORTED/INSUFFICIENT/CONTRADICTORY:
          → dialogue_manager.set_verify_result()
          → dialogue_manager.advance_to(对应节点)
```
